# 红包"颈"上添花 - 新春放松小游戏

## 游戏简介

通过摄像头实时识别身体姿态，左右移动接住掉落的红包，完成随机弹出的放松任务（举手、抬头、双手放颈后等），在游戏中放松肩颈。

## 在线访问

打开浏览器访问：**https://rita911018.github.io/neck-exercise-game/**

## 使用说明

### 环境要求

- **浏览器**：Chrome / Edge / Firefox / Safari（推荐 Chrome 或 Edge）
- **设备**：电脑（推荐） / 平板 / 手机
- **摄像头**：需要前置摄像头
- **网络**：首次加载需要网络，之后可离线游玩

### 开始游戏

1. 打开游戏页面，等待 AI 模型加载完成（页面底部会显示"准备就绪"）
2. 点击 **开始挑战**
3. 浏览器会请求摄像头权限，点击 **允许**
4. 倒计时结束后游戏开始

### 游戏玩法

- **接红包**：左右移动身体接住掉落的红包，获得分数
- **躲炸弹**：避开黑色炸弹，碰到会扣 500 分并眩晕 2 秒
- **完成任务**：游戏中会随机弹出放松任务，按提示保持姿势 3 秒即可完成
- **游戏时长**：每局 60 秒

### 任务动作

| 任务 | 动作 | 效果 |
|------|------|------|
| 左手招财 | 举起左手 | 放松左肩 |
| 右手招财 | 举起右手 | 放松右肩 |
| 步步高升 | 双手高举过头 | 拉伸肩颈 |
| 福气临门 | 双手放颈后 | 放松颈部 |
| 抬头看烟花 | 抬头朝天看 | 放松颈部 |

### 计分规则

- 普通红包：+100 分
- 金色红包：+500 分
- 舞狮红包：+2000 分（触发极速模式）
- 完成任务：+1000 分
- 碰到炸弹：-500 分

### 其他功能

- **暂停/继续**：游戏中点击左上角暂停按钮
- **查看照片**：完成任务时会自动截图，结算后可查看
- **下载照片**：点击照片放大后可下载
- **清空记录**：照片墙页面可一键清空所有记录

## 本地运行

如果需要在本地运行：

```bash
cd 个人游戏版本
python3 -m http.server 8000
```

然后浏览器打开 `http://localhost:8000`

## 常见问题

**Q: 摄像头无法打开？**
A: 检查浏览器是否允许了摄像头权限。必须通过 HTTPS 或 localhost 访问。

**Q: 手机上很卡？**
A: 手机性能有限，姿态检测帧率较低。推荐使用电脑体验。

**Q: 任务检测不灵敏？**
A: 确保全身在摄像头画面内，光线充足，动作幅度大一些。
